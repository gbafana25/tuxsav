Tuxsav

Automated backup for text files

About

Tuxsav allows for simple plain text files edited in Vim to be backed up while they are being edited. The client application does this by reading the Vim swap file, and sending that data to one of the server's endpoints.

Learn more about setuphere.

Languages/Frameworks

  • Client: C++
  • Server: Django, Python
