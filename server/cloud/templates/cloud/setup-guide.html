Getting started guide

Client setup

The C++ client reads your username and API key from a file named config.json. After compiling the client, run ./main -s in order to create and populate this file. Before a document can be saved to the server, an document object must be created by runnning ./main -c [document]. The second argument is just the name of the document on the server, and doesn't need to be the same as the name of the local file. Finally, the document can be edited with ./main -i [document_name] /path/to/file.

Server setup

The Django server can be run just like any Django app, with ./manage.py runserver. From there, you can create accounts and interact with users as normal. Consultthis guide on additional steps required if you are new to Django.
